- Business plan raises limit to **6 hours per video**; 4K output also unlocked
- Positioned as an **async video communication tool** for remote teams rather than a traditional screen recorder
- Available as a Chrome Extension, desktop app (Win/Mac), and mobile app (iOS/Android)

## 6. Fraps (Legacy)

| Property | Details |
|---|---|
| **Website** | https://www.fraps.com |
| **Developer** | Beepa Pty Ltd |
| **Platforms** | Windows ONLY (XP/Vista/7/8/10; limited Win11 support) |
| **License Model** | One-time perpetual purchase |
| **Free Trial** | Yes - free version with 30-second recording limit and watermark |
| **Pricing Type** | One-time purchase (legacy; no active development) |
| **Status** | ABANDONED - last release was February 26, 2013 (v3.5.99) |

### Pricing Tiers

| Plan | Price | Notes |
|---|---|---|
| **Free Version** | $0 | 30-second recording limit, BMP screenshots only, watermark |
| **Single User License** | $37 one-time | Full unlimited recording, all screenshot formats |
| **2-User Site License** | $70 ($35/ea) | Volume discount |
| **5-User Site License** | $150 ($30/ea) | Volume discount |
| **10-User Site License** | $250 ($25/ea) | Volume discount |
| **50-User Site License** | $950 ($19/ea) | Maximum volume discount |

### Key Features
- **Game benchmarking** - FPS counter overlay with custom benchmark logging between two points
- **DirectX and OpenGL game capture** - records video from any game using DirectX 6-12 or OpenGL
- **High frame rate capture** - up to 120 FPS recording at up to 7680x4800 resolution
- **Real-time video compression** - uses Fraps codec (lossless); very large file sizes but zero quality loss
- **Screenshot capture** - configurable hotkey, auto-named with timestamps, saved as BMP, JPG, PNG, or TGA
- **Frame rate display** - shows FPS counter in a corner of the screen during gameplay
- **Timed screenshots** - automatic screenshot capture at configurable intervals for time-lapse

### Notable Facts
- Historically the **dominant game recording tool** from the early 2000s through mid-2010s
- Now considered fully **abandonware** - last update was over 12 years ago (2013)
- Fraps recordings use an **uncompressed lossless codec** that produces extremely large files (multi-GB for short clips)
- Largely superseded by **NVIDIA ShadowPlay, Xbox Game Bar, OBS Studio, and Bandicam** which have no recording limits and smaller file sizes
- The Fraps website still functions and the purchase link is still active, but the software is no longer maintained
- May have **compatibility issues on Windows 11** due to changes in DirectX and OS security
- Beloved by retro gaming communities and still used for absolute lossless quality captures on older systems

| Property | Details |
|---|---|

| **Developer** | NVIDIA Corporation |
| **Platforms** | Windows ONLY (requires NVIDIA GeForce GPU) |
| **License Model** | Free (bundled with NVIDIA App / GeForce Experience) |
| **Free Trial** | N/A - completely free |
| **Pricing Type** | Free; hardware requirement (NVIDIA GPU) |

### Pricing Tiers

| Plan | Price | Notes |
|---|---|---|
| **NVIDIA App (includes ShadowPlay)** | $0 | Requires NVIDIA GeForce GPU; replaced GeForce Experience |

### Key Features
- **Instant Replay (DVR mode)** - continuously buffers the last 30 seconds (configurable) of gameplay in the background; save with a hotkey after something epic happens
- **Manual recording** - record at up to **8K HDR at 30 FPS** or **4K HDR at 120 FPS** with minimal GPU performance impact
- **AV1 encoding** for RTX 40-series and newer GPUs - smaller file sizes with higher quality vs H.264/H.265
- **NVIDIA Highlights** - automatically detects and saves key game moments (kills, victories, achievements) in supported titles
- **Screenshot capture** - up to 8K resolution screenshots
- **Live streaming** - broadcast directly to Twitch or YouTube in up to 1080p 60fps with two clicks
- **GPU-accelerated encoding** (NVENC) ensures minimal impact on game frame rates
- **In-game overlay** (Alt+Z) - access all recording, streaming, and performance monitoring features without leaving the game

### Notable Facts
- Completely **free** but requires an NVIDIA GeForce graphics card; AMD and Intel GPU owners cannot use it
- ShadowPlay has been **rebranded and integrated** into the NVIDIA App, which replaced GeForce Experience
- The **Instant Replay feature** is a standout - no need to have recording running; retroactively save moments that already happened
- NVIDIA Highlights uses game-specific APIs to automatically clip important moments without any user input
- One of the **lowest-overhead game recorders** available; hardware NVENC encoding has negligible impact on FPS
- RTX 40-series introduced **AV1 codec support**, delivering significantly better quality-to-file-size ratio
- Hotkeys: Alt+Z (overlay), Alt+F9 (record), Alt+F10 (save replay), Alt+F1 (screenshot)

### Key Features
- **One-click screen recording** - open overlay with Win+G, start recording with Win+Alt+R hotkey
- **Saves as MP4** - recordings saved to Videos/Captures folder automatically
- **Microphone toggle** during recording (Win+Alt+M)
- **Screenshot capture** (Win+Alt+PrtScn) directly from the overlay

- **Audio controls** - adjust volume of individual apps from the overlay without leaving your game

- The overlay is activated with **Win+G** at any time; recording starts/stops with Win+Alt+R
- Also accessible from the **Snipping Tool** in recent Windows 11 updates, which adds short-clip screen recording
- **Does not support recording the desktop** directly - must be in an app or game window for recording to work
- Hardware GPU encoding (NVENC/AMF/QSV) is used when available, reducing CPU load during recording
- Excellent for casual gamers and users who need a zero-cost, zero-install screen recorder

## 💰 Pricing Model Notes

Many vendors have **transitioned to subscription-only** models in recent years:

- **TechSmith (Camtasia & Snagit)** - Switched to mandatory annual subscriptions in 2025. Legacy perpetual licenses are honored until 2030 for existing maintenance customers, but new users cannot buy perpetual.
- **ScreenFlow** - Remains one of the few professional tools still offering **perpetual licenses**.
- **Bandicam, Icecream, Movavi, HitPaw, Wondershare** - Offer **both subscription and one-time purchase** options, giving users flexibility.
- **Loom, ScreenPal, Screencastify, Riverside** - Pure **subscription SaaS** with freemium tiers.

This shift has sparked debate in the user community, particularly around tools like Snagit where long-time users felt blindsided by the mandatory subscription change. If subscription fatigue is a concern, prioritize tools that still offer perpetual licenses or have generous free tiers.
